Bitcoin anchoring via OpenTimestamps

Commitment hashes are published anonymously to Bitcoin via public calendar servers. Each hash is one anonymous leaf in a Merkle tree — no on-chain identity links your entries. Anchoring is feeless. Confirmation arrives within a few hours as the next Bitcoin block is mined. A .ots proof file can be verified offline by any standard OpenTimestamps tool.
