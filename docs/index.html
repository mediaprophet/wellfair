wellfair — wellbeing vault

An independent, privacy-first personal wellbeing informatics vault by Timothy Charles Holborn. Explore the live browser experience powered by WebAssembly (Rust core) and Pyodide (Python analytics).

Open the browser demo

What runs in the browser?

  • Analytics and dashboards powered by Python running in Pyodide.
  • Samsung Health CSV → RDF conversion via the Rust core compiled to WASM.
  • Synthetic demo profile mirroring the Streamlit experience.

Going further

  • Run the full Streamlit UI locally: streamlit run ui/app.py
  • Upcoming native shells — Electron/Tauri desktop & Capacitor mobile.
  • All source code: README · License

Install as a PWA

On supported browsers, use “Add to Home Screen” to pin the demo. Everything runs client-side; no data leaves your device.

Need the full vault with your own data? Clone the repo, run the Streamlit app, and plug in your Samsung Health export.
