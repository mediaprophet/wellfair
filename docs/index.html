wellfair — personal welfare informatics vault

Own your wellbeing story across clinical, social, legal, and lived-context dimensions. wellfair harmonises Rust, Python, and semantic standards into a human-first vault that travels with you — across services, across crises, across victories.

Launch the in-browser experience

Layered human context

  • Maslow-informed lenses surface physiological, safety, belonging, esteem, and growth signals instantly.
  • Persona packs (housing, disability, mental health, elder advocacy) arrive pre-curated for demonstrations.
  • Delegation rules ensure the right advocate receives the right slice of context with explicit consent.

Powerful local analytics

  • Rust core compiled to WebAssembly processes Samsung Health exports entirely client-side — no cloud hop.
  • Pyodide drives advanced wellbeing analytics and storytelling dashboards directly in the browser.
  • Streamlit desktop experience unlocks sanctuary mode, case orchestration, and rich ontology authoring.

Build beyond the demo

  • Run locally: pip install -r requirements.txt && streamlit run ui/app.py
  • Compile the Rust core: wasm-pack build --target web --out-dir docs/pkg
  • Dive into the architecture: README · philosophy · Sanctuary mode

Install as a trusted companion

On supported browsers, tap “Add to Home Screen” to keep wellfair one gesture away. Every calculation, every RDF export, every analytics pulse stays on your device.
