health-to-solid

health-to-solid is a local Personal Wellbeing Informatics Vault and RDF export pipeline developed independently by Timothy Charles Holborn.

This documentation site provides an introduction and a lightweight browser demo that runs the Rust-based core compiled to WebAssembly. All files you upload to the demo are processed locally in your browser and are not transmitted to any server.

Install on mobile

On supported mobile browsers, you can add the demo to your home screen via the browser menu. The demo runs entirely client-side.
